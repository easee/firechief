# FireChief Bot Resources

This directory contains assets and configuration text for the FireChief bot integrations.

## Contents

### Bot Descriptions
- `slack-bot.md` - Slack bot app configuration, descriptions, and setup text
- `notion-integration.md` - Notion integration configuration and descriptions

### Assets
- `fire_chief_bot.png` - Bot logo/icon
- `fire_chief_bot.svg` - Vector version (optional, for scalability)

## Usage

### For Slack App Setup
1. Create new Slack app at https://api.slack.com/apps
2. Use text from `slack-bot.md` for app descriptions
3. Upload `fire_chief_bot.png` as app icon
4. Copy the bot scopes and permissions listed

### For Notion Integration
1. Create new integration at https://www.notion.so/my-integrations
2. Use text from `notion-integration.md` for integration description
3. Upload `fire_chief_bot.png` as integration logo
4. Follow database setup instructions

### For Documentation
When writing documentation or README files:
- Reference these files for consistent messaging
- Use the same descriptions across all platforms
- Keep version-controlled for team consistency

## Updating Descriptions

If you improve the bot descriptions:
1. Update the relevant `.md` file in this directory
2. Update the live Slack app and Notion integration
3. Commit changes so the team has the latest version
